---
- OS we are using: Ubuntu 19.04
- Prerequisites:
- H/W:
- 256 MB RAM, 512 MB recommended
- 10 GB Hard Disk
- S/W:
- Java8:
- sudo apt-get update
- sudo apt-get install openjdk-8-jre
- Docker CE:
- Add the GPG key:
- curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
- Add repository:
- sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable edge test"
- Note that the stable version of docker ce for Ubuntu 19.04 doesn't exist as of this writing
- Hence, we had to use edge/test version
- Ensure installing docker from the repository added above, and not from the ubuntu default repo:
- apt-cache policy docker-ce
- Install Docker:
- sudo apt-get install docker -y
- verify installation: docker version
- Install Jenkins:
- Download Jenkins: wget http://mirrors.jenkins.io/war-stable/latest/jenkins.war
- Compile with Java and assign a web port: java -jar jenkins.war --httpPort=8080
- Jenkins should be running now on port 8080
- To access Jenkins, use the IP and port on the browser or curl it
- It can be accessed via http://localhost:8080 or http://<ipaddress>:8080
- Following the instructions on the screen to complete the installation
- In my case the Administrator password was found in /home/networkandcode/.jenkins/secrets/initialAdminPassword
- That had to be entered to proceed with the installation
- I have chosen to Install Jenkins with suggested Plugins (recommended by the community)
- Examples of Plugins are Git, Gradle, LDAP
- Set up admin credentials, name and email
- you can change the URL if required
- The setup is now complete
- It shows the complete Jenkins dashboard, and new jobs could now be created
